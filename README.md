# Woord van de Dag

An Android application that helps users learn advanced Dutch vocabulary (C1/C2 level) through daily words.

## Features

- Daily Dutch word with pronunciation, definition, and example usage
- Push notifications for new words
- History of previously shown words
- Clean Material Design UI
- Offline support with local word database

## Technical Details

- Built for Android API level 24+
- Uses Kotlin
- Implements MVVM architecture
- Uses AndroidX libraries
- WorkManager for scheduling notifications
- ViewBinding for view handling

## Setup

1. Clone the repository
2. Open in Android Studio
3. Sync project with Gradle files
4. Run on an Android device or emulator
